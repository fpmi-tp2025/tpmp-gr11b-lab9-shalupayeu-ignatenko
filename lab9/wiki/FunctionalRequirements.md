# Functional Requirements

## Authentication
- Login with username and password
- Session persistence using UserDefaults
- Logout functionality

## Accounts
- Display list of accounts
- Show only active and blocked accounts
- Hide closed accounts
- Account types:
  - Current
  - Savings
  - Credit
  - Card account
- Card account types:
  - Salary
  - Savings
  - Credit
- Overdraft support for salary accounts

## Additional Features
- Currency exchange rates view
- Map with bank branches
- Find nearest branch using GPS

## Non-functional behavior
- No crash on invalid input
- Offline access for accounts
- Online access required for currency rates
