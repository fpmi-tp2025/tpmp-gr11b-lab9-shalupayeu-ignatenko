# 3.1 Функциональные требования

## Авторизация
- вход по логину/паролю
- сохранение сессии (UserDefaults)
- выход из системы

## Счета
- просмотр списка счетов
- отображение только активных и заблокированных
- скрытие закрытых счетов
- типы счетов:
  - текущий
  - сберегательный
  - кредитный
  - карт-счет
- типы карт-счетов:
  - зарплатный
  - сберегательный
  - кредитный
- поддержка овердрафта для зарплатных счетов

## Дополнительно
- просмотр курса валют
- карта отделений
- поиск ближайшего отделения
